# Файлообменник
FileShare - web-приложение для загрузки, хранения и обмена файлами между пользователями на базе Flask и SQlite3

## Технологии
- [sqlite3] - встроенная python-библиотека для работы с базами данных
- [flask] - простой python-фрэймворк для создания web-приложений

## Функционал
### Основные возможности
1. Загрузка файлов через Drag-and-Drop.
2. Создание временных ссылок.
3. Создание лимита по скачиваниям.
4. Защита файлов паролем.
### Безопасность
1. Проверка типов файлов.
2. Ограничение размера файлов.

## Использование
### Загрузка файлов
1. Перейдите на главную страницу.
2. Перетащита файл в область загрузки.
3. Настройте параметры доступа.
4. Получите ссылку для скачивания.
### Скачивание файлов
1. Перейдите по полученной ссылке.
2. Введите пароль, если требуется.
3. Скачайте файл.

## Требования
- Python 3.8+
- Flask
- SQLite3 (встроен в Python от версии 3.8+)
