# Генератор паспортных данных
Сайт для генерации случайных, не существующих паспортных данных(серия, номер, ФИО, пол, дата и место рождения) для паспортов разных стран.


## Технологии
- [random] - python-библиотека для "случайного" выбора
- [flask] - простой python-фрэймворк для создания web-приложений

## Использование
Для использования сайта необходимо авторизироваться!
1. Выберите формат(страну) паспорта
2. Выберите необходимое количество генерируемых данных
3. Если есть предпочтения для генерируемых данных - выберите нужные, если нет - оставьте случайными
4. Выберите формат вывода данных (построчно или списком)
5. Нажмите кнопку "Сгенерировать"
6. Скопируйте полученные данные
